Ответы на Частозадаваемые Вопросы и API

Почему парсинг фильмов – это законно?

Парсинг (веб-скрейпинг) — это автоматизированный сбор общедоступной информации из интернета, осуществляемый без использования API сайтов. Можно представить, как человек открывает браузер, ходит по сайтам и копирует с них данные. Парсинг — то же самое, только ходит не человек, а робот.

Согласно 29 статье Конституции РФ – каждый вправе свободно искать и получать информацию любым законным способом. И Закон об информации гласит, что информация является общедоступной, и каждый вправе использовать такую информацию (при указании источника).

Также Совет безопасности РФ одобрил использование пиратского контента в условиях санкций. То есть все иностранные фильмы и сериалы стали общедоступными и перестали обладать авторским правом. На сайте нельзя смотреть русские фильмы, так как это бы нарушало 1263 статью ГК РФ отечественных кинокомпаний.

Получить доступ к API и его использование

Для начала выберите подходящий для себя тариф и получите Ключ разработчика.

Получить прямую ссылку на фильм / сериал

Формат запроса:

{{ request.host_url }}api
? type = <string>
& server = <string>
& code = <string>
& season = <integer>
& episode = <integer>
& apikey = <string>

// Фильм или Сериал [film, serial]
// Сайт [kinovod, rutube, zona]
// Код фильма / сериала на сайте
// Сезон (только для сериала)
// Серия (только для сериала)
// Ключ Разработчика

Формат ответа:

{
  "src": <string>,
  "seasons": <integer>,
  "episodes": <integer>,
  "error": <string>
}

// Ссылка на исходное видео
// Количество сезонов (только в сериале)
// Количество серий в сезоне (только в сериале)
// Код ошибки и описание (при возникновении)
